# 启动软件
1. 双击 `python.exe` 即可启动
2. 启动后会在系统托盘显示绿色圆形图标
3. 桌面中央会出现哈罗宠物

# 系统托盘菜单功能
- 跟随指针: 开启/关闭哈罗跟随鼠标
- 🗣️ 打招呼: 手动触发哈罗的问候
- 👤 用户设置: 自定义用户名和问候内容
- ℹ️ 关于: 查看软件信息
- ❌ 退出: 退出软件

# 交互方式
1. 点击哈罗: 直接与哈罗互动，会触发问候，点击两下左右摇晃，点击三下是转身
2. 拖拽窗口: 移动哈罗到想要的位置
3. 右键托盘: 打开功能菜单

# 技术信息

# 依赖库
- PyQt5 (GUI框架)
- Python 3.7+


# 注意事项

1. 防病毒软件: 某些防病毒软件可能误报，这是PyQt5程序的常见情况
2. 首次运行: 可能需要几秒钟的启动时间（解压内置资源）
3. 权限要求: 无需管理员权限即可运行
4. 单实例: 软件会自动检测，避免重复运行

# 故障排除

# 如果哈罗没有出现
1. 检查系统托盘是否有绿色图标
2. 尝试重启软件
3. 检查是否有错误日志

# 如果用户设置打不开
1. 确保只有一个实例在运行
2. 尝试以管理员权限运行
3. 检查系统托盘菜单连接

# 如果软件无法启动
1. 检查系统是否支持.NET Framework
2. 尝试更新显卡驱动
3. 查看系统事件日志
